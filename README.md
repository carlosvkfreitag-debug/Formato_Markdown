# Markdown: Historia, utilidad y casos de uso

## 1) ¿Qué es Markdown?
Markdown es un lenguaje de marcado sencillo que se utiliza para dar formato a textos de manera fácil y rápida.

## 2) Origen e historia de Markdown
- Lo creó John Gruber.
- Lo creó en 2004.
- Fue creado para permitir que las personas pudieran escribir texto con formato de manera sencilla y legible, usando texto plano, y que ese texto pudiera       convertirse fácilmente en HTML.

## 3) ¿Para qué sirve Markdown?
Sirve para escribir textos que luego pueden convertirse en HTML u otros formatos, sin necesidad de usar código complicado. Es muy usado para documentación, blogs, notas y repositorios de programación.
  
## 4) Casos de uso reales
GitHub
Se utiliza para escribir archivos README.md, documentar proyectos, explicar cómo usar un programa y colaborar entre desarrolladores.

Documentación técnica
Muchas guías, manuales y documentación de software se escriben en Markdown por ser claro, ordenado y fácil de mantener.

Blogs y páginas web
Plataformas como generadores de sitios estáticos permiten escribir artículos en Markdown y convertirlos automáticamente en páginas web.

Notas personales y académicas
Estudiantes y profesionales lo usan para tomar apuntes, organizar ideas y crear documentos simples sin usar procesadores de texto complejos.

## 5) Ventajas y desventajas

### Ventajas
-Es fácil de aprender y usar, ya que utiliza símbolos simples.

-Permite escribir texto rápido y ordenado sin necesidad de conocimientos avanzados de programación.

### Desventajas
-Tiene opciones de formato limitadas en comparación con editores como Word.

-No existe un estándar único, por lo que algunas funciones pueden variar según la plataforma.

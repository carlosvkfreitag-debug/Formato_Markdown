# Markdown: Historia, utilidad y casos de uso

## 1) ¿Qué es Markdown?
Markdown es un lenguaje de marcado sencillo que se utiliza para dar formato a textos de manera fácil y rápida.

## 2) Origen e historia de Markdown
- Lo creó John Gruber.
- Lo creó en 2004.
- Fue creado para permitir que las personas pudieran escribir texto con formato de manera sencilla y legible, usando texto plano, y que ese texto pudiera       convertirse fácilmente en HTML.

## 3) ¿Para qué sirve Markdown?
Sirve para escribir textos que luego pueden convertirse en HTML u otros formatos, sin necesidad de usar código complicado. Es muy usado para documentación, blogs, notas y repositorios de programación.
